    Before : 한 소설 전체를 입력
    After : 연재 형태로 매 회차를 입력 

2. 일반 독자와 관리자(작가) 화면 분리
- 일반 독자 화면
    - 모든 작품들의 리스트를 확인할 수 있고 각 작품 클릭 시 간단 설명과 현재까지의 모든 회차들이 리스트업됨.
        - 여기에 편의성 (봤던 작품 우선 순위 높인다던가 최근 업로드 우선 순위 높인다거나)도 기획해
    - 각 회차 클릭 시, 전체화면으로 비주얼 노벨이 보이며 ESC나 버튼 클릭으로 비주얼 노벨 나가기 기능 지원
    - 한번 본 회차는 회색 처리?
- 관리자(작가) 화면
    - 자신의 작품들만 확인 할 수 있음
    - 작품 클릭하면 상세로 넘어가고 각 회차 텍스트들이 존재. 
        - 작품의 캐릭터, 배경화면 에셋들을 확인 할 수 있음.
    - 각 회차의 내용은 수정 불가. 한번 올리면 끝. (삭제는 가능)
    - 무조건 다음 회차만 추가 가능
    - 작품 업로드 시 현재 모든 pipeline 돌려서 최종 형태 비주얼 노벨까지 뽑아냄. (BGM 제외)
- 정확히는 누구나 독자 혹은 작가가 될 수 있으니 "나의 작품" 이란 탭을 누르면 관리자 화면으로 이동하면 좋을 듯.
